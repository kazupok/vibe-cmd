# テスト用AIプロンプト集

## テストケース生成プロンプト

### ユニットテストケース生成
```
以下の関数/クラスのテストケースを網羅的に生成してください：

[コード]

テストケースに含めるべき項目：
1. 正常系
   - 典型的な入力
   - 境界値
   - 特殊な有効値

2. 異常系
   - null/undefined
   - 空文字/空配列
   - 型の不一致
   - 範囲外の値

3. エッジケース
   - 最大/最小値
   - 特殊文字
   - 並行処理

各テストケースには以下を含めてください：
- テストケース名（日本語）
- 入力値
- 期待される結果
- テストの意図
```

### 統合テストシナリオ生成
```
以下の機能の統合テストシナリオを作成してください：

機能: [機能名]
関連コンポーネント: [コンポーネントリスト]

シナリオに含めるべき内容：
1. 正常フロー
   - ユーザーの一連の操作
   - システムの応答
   - データの整合性確認

2. 異常フロー
   - エラー発生時の動作
   - リトライ処理
   - ロールバック

3. 性能要件
   - レスポンスタイム
   - 同時接続数
   - データ量

各シナリオは以下の形式で記述：
- 前提条件
- 操作手順
- 期待結果
- 確認項目
```
